# /plan Command

When you run `/plan`, the assistant will:
1. **Research** the current context of the task.
2. **Activate** the `blueprint` skill.
3. **Generate** a phased implementation plan.
4. **Wait** for your "Yes" or "Proceed" confirmation.

> [!IMPORTANT]
> Aplan is NOT an implementation. It is a roadmap. Do not start coding until the plan is green-lit.
